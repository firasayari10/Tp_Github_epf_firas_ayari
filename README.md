## Choix techniques
- Python 3.11
- `dataclasses` pour le modèle Student
- `StudentService` pour la logique métier
- CLI minimal pour interagir avec les étudiants
- Export CSV et JSON pour la sauvegarde des données

## Commandes Git utilisées
- `git clone`, `git branch`, `git checkout`
- `git add`, `git commit`, `git push`
- `git fetch`, `git merge`, `git rebase`
- Gestion des conflits avec `<<<<<<< HEAD` / `>>>>>>> branch`
- Suppression de branche locale et distante : `git branch -d` / `git push --delete`
- Tagging : `git tag -a v1.0 -m "Version 1.0"`
